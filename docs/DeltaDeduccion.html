Derivación de fórmulas proposicionales

derivP :: FProp -> SimboloProposicional -> FProp Source #

(derivP f v) es la derivada de la fórmula proposicional f respecto de la variable v. Por ejemplo,

>>> derivP (p ∧ q → r) "p"
¬(q ↔ (q ∧ r))
>>> derivP (p ∧ q → r) "q"
¬(p ↔ (p ∧ r))
>>> derivP (p ∧ q → r) "r"
(p ∧ q)
>>> derivP (p ∧ q → p ∨ q) "p"
⊥

sustituir :: FProp -> SimboloProposicional -> FProp -> FProp Source #

(sustituir f v g) es la fórmula obtenida sustituyendo en la fórmula f la variable v por la fórmula g. Por ejemplo,

>>> sustituir (p → q ∨ r) "q" (p ∧ q)
(p → ((p ∧ q) ∨ r))

variablesProp :: FProp -> [SimboloProposicional] Source #

(variablesProp f) es la lista de las variables proposicionales de la fórmula f. Por ejemplo,

>>> variablesProp (p → q ∨ p)
["p","q"]

prop_derivP_sustituir :: FProp -> Bool Source #

Comprueba que, para toda fórmula f y toda variable x de f, la derivada de f respecto de x es equivalente a la fórmula ¬(f[x/¬x] ↔ f)

La regla de independencia (regla delta)

indep :: Polinomio -> Variable -> Polinomio Source #

(indep p x) es el polinomio formado por los monomios de que no contienen la variable x. Por ejemplo,

>>> indep (P[M["x","y"],M["y","u"],M ["z"]]) "x"
y*u+z

delta :: Polinomio -> Polinomio -> Variable -> Polinomio Source #

(delta a1 a2 v) es el polinomio obtenido aplicando la regla delta (o de independencia) a los polinomios a1 y a2 respecto de la variable v; es decir, el polinomio 1 + (1+a1*a2)*(1+a1*c2+a2*c1+c1*c2) donde ci es la derivada de ai respecto de v (para i = 1, 2). Por ejemplo,

>>> delta (P[M["x","y"],M["y","z"]]) (P[M["u","x"],M["y","z"]]) "x"
u*y+u*y*z+y*z

delta' :: Polinomio -> Polinomio -> Variable -> Polinomio Source #

(delta' a1 a2 v) es el polinomio 1 + (1+b1*b2)*(1+(b1+c1)*(b2+c2)) donde ci es la derivada de ai respecto de v y bi es es el polinomio formado por los monomios de ai que no contienen la variable x (para i = 1, 2). Por ejemplo, 

>>> delta' (P[M["x","y"],M["y","z"]]) (P[M["u","x"],M["y","z"]]) "x"
u*y+u*y*z+y*z

deltaP :: FProp -> FProp -> SimboloProposicional -> FProp Source #

(deltaP f1 f2 v) es la f§ormula obtenida aplicando la regla delta (o de independencia) a las fórmulas f1 y f2 respecto de la variable v. Por ejemplo,

>>> deltaP (r → p ∨ q) (q → p ∧ q) "q"

derivadas :: [Polinomio] -> Variable -> [Polinomio] Source #

(derivadas ps x) es la lista de los polinomios obtenidos aplicando la regla delta a dos polinomios de ps respecto de la variable x. Por ejemplo,

>>> derivadas [P[M["x","y"],M["y","z"]],P[M["a","y"],M["y"]]] "y"
[x+z,a*x+a*z+x+z,1+a]

derivadasP :: [FProp] -> SimboloProposicional -> [FProp] Source #

(derivadasP fs x) es la lista de las proposiciones obtenidas aplicando la regla deltaP a dos fórmulas de fs respecto de la variable x. Por ejemplo,

>>> derivadasP [p → q ∨ r, r → p] "q"
[¬(⊤ ↔ ¬((p ∧ r) ↔ r))]
>>> derivadasP [p → q ∨ r, r → q] "q"
[]
