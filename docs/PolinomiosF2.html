Productos de polinomios

productoMM :: Monomio -> Monomio -> Monomio Source #

(productoMM m1 m2) es el producto de los monomios m1 y m2. Por ejemplo,

>>> productoMM (M ["x","y"]) (M ["y","z"])
x*y*z

productoMP :: Monomio -> Polinomio -> Polinomio Source #

(productoMP m p) es el producto del monomio m por el polinomio p. Por ejemplo,

>>> productoMP (M ["x","y"]) (P [M ["a"], M ["y","z"]])

producto :: Polinomio -> Polinomio -> Polinomio Source #

(producto p1 p2) es el producto de los polinomios p1 y p2. Por ejemplo,

>>> producto (P [M ["x","y"], M ["z"]]) (P [M ["a"], M ["y","z"]])
a*x*y+a*z+x*y*z+y*z
>>> producto (P [M ["x","y"], M ["z"]]) (P [M ["x"], M ["y","z"]])
x*y+x*y*z+x*z+y*z
>>> producto (P [M ["x"], M ["y"]]) (P [M ["x"], M ["y"]])
x+y

Derivada de polinomios

deriv :: Polinomio -> Variable -> Polinomio Source #

(deriv p x) es la derivada del polinomio p respecto de la variable x; es decir, la lista de monomios de p que contienen la variable x, eliminándola. Por ejemplo,

>>> deriv (P [M ["u"],M ["x"],M ["x","y"],M ["x","z"]]) "x"
