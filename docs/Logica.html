Sintaxis de fórmulas proposicionales

type SimboloProposicional = String Source #

Los símbolos proposicionales se representan por cadenas.

data FProp Source #

FProp es el tipo de las fórmulas proposicionales definidas por

  • T y F son fórmulas
  • Si A es una fórmula, también lo es ¬A.
  • Si A y B son fórmulas, entonces (A ∧ B), (A ∨ B), (A → B) y (A ↔ B) también lo son.

Semántica de la lógica proposicional

type Interpretacion = [FProp] Source #

Las interpretaciones son listas de fórmulas atómicas. Las fórmulas de las interpretaciones se suponen verdaderas y las restantes fórmulas atómicas se suponen falsas.

significado :: FProp -> Interpretacion -> Bool Source #

(significado f i) es el significado de la fórmula f en la interprestación i. Por ejemplo, 

>>> significado ((p ∨ q) ∧ ((no q) ∨ r)) [r]
False
>>> significado ((p ∨ q) ∧ ((no q) ∨ r)) [p,r]
True
